# ECE CK801 - Advanced Programming Techniques

Welcome to our project repository! 

We are a team of three undergraduate students from the **University of Patras, Greece**, currently in our **8th semester** of studies in Electrical and Computer Engineering. 

This repository contains all the labs and the main project we are implementing as part of the *Advanced Programming Techniques (CK801)* course.

---

## The Project

Our main project focuses on building a **Smart Waste Management System**. 
By leveraging **Raspberry Pi** and **Python**, we are developing a system designed to effectively monitor and manage waste disposal, turning standard bins into IoT-enabled smart devices!

### Repository Structure

* **`/labs`**: Contains the code and reports for our weekly lab assignments.
* **`/smartWasteBin`**: Contains our smart waste bin project (moved to a new repo).

## Getting Started

To get started with the project, clone the repository to your local machine and navigate into the project directory:

```bash
git clone https://github.com/jimfil/raspberryPiProject.git
cd raspberryPiProject
```
