# Smart Waste Bin Project

This is the repository for the Smart Waste Bin course project. 
It integrates a `HC-SR501` PIR motion sensor on a Raspberry Pi to create a concurrent data acquisition and processing pipeline.

## Milestones Achieved
- **Milestone 1:** Basic structure, GitHub integration, reproducible environments.
- **Milestone 2:** Sensor integration & `JSONL` outputs via Python queue limits.
- **Milestone 3:** Extract Transform Load (ETL) pipeline using thread concurrency, queue backpressure, decoupled sensing and storage.

## Installation
Use a python `venv` and install the requirements:
```bash
pip install -r requirements.txt
```
